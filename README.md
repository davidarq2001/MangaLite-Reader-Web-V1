MangaLite Reader Web V1

I designed this web version to facilitate reading manga locally without having to resort to an online site or the Windows photo viewer. It's more convenient, visually appealing, perfect for what's needed, 
and above all, easy and fast.

Features:
- Day/Night Mode
- Reads PDF files, individual files, and image folders
- Contact information button
- Zoom control
- Page turning with the arrow keys

Details:
- Displays the name of the open file or folder
- Displays the number of pages, total pages, and current page
